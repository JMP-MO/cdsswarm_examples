### Juptyer notebook - how to download ERA5 in parallel. 
I have provided a jupyter notebook demonstrating how you could use the `cdsswarm` package to download ECMWF ERA5 data in parallel. This includes taking 2 separate jobs (surface and pressure levels) and then splitting these into multiple tasks by month and year for running in parallel.
